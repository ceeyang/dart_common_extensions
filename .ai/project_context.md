# Project Context: dart_common_extensions

## Project Overview
A comprehensive collection of Dart extensions for `DateTime`, `List`, `Map`, `String`, `num`, and `Enum`.

## Current Phase
Adding `toMoney` extension methods to common types (`num`, `String`).

## Tech Stack
- Dart SDK: >=3.0.0 <4.0.0
- Dependencies: `intl`, `decimal`

## Tooling & Conventions
- Unit tests using `package:test`.
- Linting using `package:lints`.
- Documentation with doc comments and examples.
- Extension methods organized by type in `lib/extensions/`.
